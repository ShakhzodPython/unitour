## Prerequisites

Make sure you have Bun installed. If not, you can install it with:

```bash
curl -fsSL https://bun.sh/install | bash
source ~/.bashrc  # or ~/.zshrc depending on your shell
```
# Install Dependencies
```
bun install
bun run dev
bun dev build
```
